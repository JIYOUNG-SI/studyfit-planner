# StudyFit Planner

AI 기반 학습 계획 관리 웹앱

현재 구현 완료
- 과제 추가
- 수정
- 삭제
- 추천 학습
- 집중 모드
- 진행률 계산

추가 개발 예정
- 학습 통계(Stats)
- 학습 패턴 분석
- 과목별 그래프

## 개발 과정

### 발견된 문제

- Stats 탭에서 학습 통계가 출력되지 않음

### 원인 분석

- localStorage 저장 정상
- 데이터 로딩 정상
- 통계 화면 미구현 확인

### 향후 개선 계획

- Chart.js 기반 통계 시각화
- 학습 시간 분석
- AI 학습 추천 고도화

## 배포

Netlify를 이용하여 배포
